# การคาดคะแนยอดขาย

สำหรับวิธีการคาดคะเนนั้น ทางเราใช้ Exponential Smoothing ในการคาดเดา

Exponential Smoothing เป็นรูปแบบหนึ่งของการพยากรณ์ที่ให้ความสําคัญกับข้อมูลเก่าทุกค่า โดยให้ความสําคัญแก่ค่าที่ใกล้ปัจจุบันมากที่สุด ลดหลั่นลงไปจนถึงค่าที่ 1 และถ่วงน้ำหนักข้อมูล โดยใช้สัมประสิทธิ์การปรับเรียบ alpha (α\alphaα)

Ft=Ft−1+α(At−1−Ft−1)F_t = F_{t-1} + \alpha(A_{t-1} - F_{t-1} ) F​t​​=F​t−1​​+α(A​t−1​​−F​t−1​​)

โดยที่

  • FtF_tF​t​​ คือค่าทำนาย
  • Ft−1F_{t-1}F​t−1​​ คือค่าที่ทำนายในอดีต
  • At−1A_{t-1}A​t−1​​ คือค่าจริงในอดีต
  • α\alphaα คือสัมประสิทธิ์การปรับเรียบ (Weight) โดยที่ 0<=α<=10 <= \alpha <= 10<=α<=1

และหาค่าความคลาดเคลื่อนของข้อมูลที่ได้จากการทำนายกับข้อมูลที่เกิดขึ้นจริง ด้วย Sum of squared errors (SSE)

SSE=∑i=1N(Xi−X^i)2\text{SSE} = \sum_{i=1}^N(X_i - \hat{X}_i)^2 SSE=​i=1​∑​N​​(X​i​​−​X​^​​​i​​)​2​​

  • XiX_iX​i​​ เซตของข้อมูลที่เกิดขึ้นจริง
  • X^i\hat{X}_i​X​^​​​i​​ คือเซตข้อมูลที่ได้จากการทำนาย
  • NNN คือจำนวนข้อมูล

โปรแกรม Point of Sales (POS) Systems สามารถพยากรณ์ยอดขายให้ผู้ใช้ระบบ สามารถดูการพยากรณ์ยอดขายของวันถัดไป และเดือนถัดไปได้ โดยใช้หลักการของ Exponential Smoothing ร่วมกับค่า Sum of squared errors (SSE\text{SSE}SSE)

จากหลักการพยากรณ์ของ Exponential Smoothing จะต้องมีการเลือกค่าสัมประสิทธิ์การปรับเรียบ (α\alphaα) ที่เหมาะสม ซึ่งโปรแกรม POS จะทำการเลือกค่า ที่เหมาะสมจากการคำนวนค่า Sum of squared errors โดยจะเลือกค่า α\alphaα ที่มีค่า Sum of squared errors ที่น้อยที่สุด
